### Stage 3：Prompt生成（H1-H5 + D1-D9）

- camera（镜头）

- composition（构图）
- color control（必须 HEX）

模块如下：

每条 prompt 必须：

5. 使用 HEX 色值控制视觉风格
6. 不允许新增产品配件
7. 不允许科幻/幻想元素
8. 必须写清：
   - camera angle
   - lighting setup
   - composition

9. 输出 JSON 格式，不要只输出 markdown
